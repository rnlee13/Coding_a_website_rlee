This website is about web design created by Robin Lee as a student project for ARTD 245: Web Design at North Central College in Naperville, IL, in 2023. All the text comes from the following articles located on the Dribbble website. All the images were created by me using Photoshop. The website has been created using HTML and CSS in Dreamweaver.

Dribble: Choosing typography for website by Renee Fleck

Dribble: Choosing colors for web design by Renee Fleck

Dribble: 11 Principles of Design by Renee Fleck

Product Description

Design and develop a visually pleasing, interactive website with multiple HTML pages, navigation connecting all the pages, featuring a cohesive color scheme, a clear visual hierarchy, a balanced layout, exciting visuals, and text. The goal is to learn how to code/develop an interactive website by applying design principles.

Enjoy -Robin Lee
